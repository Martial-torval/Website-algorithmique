Les variables

Qu'est ce qu'une variable ?

L’algorithmique est une méthode de résolution de problèmes. Un algorithme consiste en une suite d’instructions non-ambigus qui permet d’arriver à un résultat donné. On peut y rapprocher la notion de « Pathfinder », la recherche du chemin optimal (dans la résolution d’un problème).

Syntaxe d'une variable

Afin d’executer un algorithme, des informations (valeurs) sont stockées puis utilisées. On utilise des variables pour stocker ces informations.
Une variable est constituée de quatre caractéristiques : un nom, un type, une valeur et une affectation.
type nom ← valeur
On peut assimiler le concept de variable à une boite. La nom de la variable représente l’étiquette et sa valeur représente le contenu de la boite.

Valeur d'une variable

La valeur d’une variable peut varier dans l’algorithme.

Nom d'une variable

Le nom de la variable est écrit sous la forme d'une chaîne de caractères constituée de lettres et de chiffres mais ne ne peut pas contenir de caractères spéciaux.
Le nom de la variable doit être explicite et le plus représentatif possible du contenu de celle-ci pour faciliter la lecture de l'algorithme.
Il ne doit pas être trop long pour ne pas nuire à la lisibilité de l’ensemble.
Il doit être différent des valeurs interdites (termes ayant déjà une signification dans l’algorithmique, par exemple « faux » ou « entier » sont des valeurs interdites).
Le nom ne commence jamais par une majuscule, ne doit pas contenir d’espace et doit utiliser la notation camelCase (une Majuscule à chaque début de mot) afin de faciliter la lecture.

Exemple: entier NumeroDeTelephone ← 0606060606

Type d'une variable

Les types définissent les caractéristiques des valeurs de la variable. Il existe quatre types primitifs de variable: Nombre entier (noté entier) : tout nombre entier, ex: 20 Nombre décimal (noté decimal) : tout nombre décimal, ex: 1,2 Caractère (noté car) : un caractère, doit se noter entre ‘cote’, ex: ‘a’ Booléen (noté booleen): seulement deux valeurs possibles: vrai ou faux. (les types primitifs s’écrivent tout en minuscule) Il existe d’autres types de variables dérivés appelés types non-primitifs. La chaîne de caractère (Chaîne): contient plusieurs caractères, doit se noter entre « guillemets »

exemple: « Bonjour ! »

L'affectation

L'affectation suit la déclaration. Il s'agit d'attribuer des valeurs ou des expressions à une variable. Lorsque que la variable est affectée, on dit que la variable vaut une valeur.


exemple: La variable NumeroDeTelephone vaut 0606060606

Les variables dont la valeur est inéchangeable dans l’algorithme sont des Constantes. Mis a part leur valeurs inchangeables, elle ont les mêmes caractéristiques que les variables. La première lettre du nom d’une constante s’écrit en majuscule.
exemple: decimal Pi ← 3.14

Opération de variable

les opérations valides sur les variables sont l’addition (+), la soustraction (-), la multiplication (x) et la division (/). Les opérations sont valides seulement les variables de même type.

exemple : entier a ← 10 + 20 // résultat vaut 30
entier b ← 10 + « hello » // Erreur, opération impossible car " hello " est une chaîne

Concaténation

la concaténation sert à insérer des chaînes de caractères ou des valeurs de variables de types Chaine les unes à la suite des autres. Elle se note avec un point.


exemple: afficher («Mon nom est ».prenom. « ! »).
